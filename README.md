# VPN-for-security
برقراری امنیت شبکه ای برای رفع نیازهای امنیتی و عدم برقراری ارتباط بدون امنیت برای خانواده و معامله گران ارز دیجیتال 

از اونجایی که کانفیگ های پولی که توسط من و خانواده استفاده میشد و اکثرشون بدون امنیت یا با امنیت حداقل بودن تصمیم گرفتم به عنوان اولین پروژه بعد از قبول شدن در رشته مهندسی کامپیوتر و شروع به یادگیری شبکه و امنیت یک سرویسی تحت عنوان وی‌پی‌ان برای امنیت بسازم که این مشکل رو رفع کنم
