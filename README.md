# VPN-for-security
برقراری امنیت شبکه ای برای رفع نیازهای امنیتی و عدم برقراری ارتباط بدون امنیت برای خانواده و معامله گران ارز دیجیتال 

از اونجایی که اکثر کانفیگ های پولی از پروتکل های بدون tls و cryptography استفاده میکنند همچنین ممکنه مورد جاسوسی قرار بگیریم شروع به یادگیری و ساخت یک VPN شخصی برای خود و خانواده کردم که علاوه بر رفع نیاز اتصال به بازار های مالی خارجی امنیت اطلاعات مهمشون هم برقرار بشه.
